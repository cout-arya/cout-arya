#### 🥗 [FuelIQ](https://github.com/cout-arya/fueliq) — AI-Powered Indian Nutrition Coach
> *MERN · Google Gemini · Pinecone · RAG · Docker · GitHub Actions*

Most fitness apps are built for Western diets. FuelIQ isn't.

- 🇮🇳 Built a **RAG pipeline** (Gemini 1.5 Flash + Pinecone) that natively understands Indian cuisine — users log meals in **Hinglish** and get accurate macro breakdowns instantly
- 📅 Smart **meal planning engine** that generates personalized plans from BMR/TDEE, cuisine region, and budget — with endpoints to swap individual meals without regenerating the entire day
- ⚙️ Production-grade **CI/CD** with GitHub Actions + Docker Compose; all external AI calls mocked with `nock` for fast, isolated, cost-free tests

#### 📄 [JDFit](https://github.com/cout-arya/jdfit) — AI-Powered ATS Resume Analyzer
> *MERN · Meta Llama 3.3 70B · OpenAI Embeddings · RAG · Docker · GitHub Actions*

ATS tools that only do keyword matching miss the point. JDFit doesn't.

- 🔍 Custom **RAG pipeline** using OpenAI Embeddings + Cosine Similarity to semantically map a resume against a JD — evaluating conceptual fit, not just keywords
- 📊 **Multi-dimensional ATS scoring** across 5 axes: Semantic Alignment, Skills, Experience, Readability, and Seniority — powered by Meta Llama 3.3 70B with zero-shot prompting
- 🔬 Combines **LLM skill extraction**, fuzzy keyword matching, and Regex impact detection to produce interpretable scores, not black-box percentages

#### 🎮 [QuizMaster.AI](https://github.com/cout-arya/quizmaster-ai) — Real-Time Multiplayer Quiz Platform
> *MERN · Socket.io · OpenAI API · WebSockets · Docker · GitHub Actions*

Real-time quizzing with AI question generation and live academic integrity enforcement.

- 🤖 **AI quiz engine** that generates contextually accurate questions from uploaded PDFs via a custom RAG pipeline — complete quizzes in seconds, not hours
- 🛡️ Real-time **academic integrity system** detecting screenshot attempts, enforcing fullscreen mode, and tracking tab switches — violations pushed live to the host dashboard via WebSockets
